# Elenty.app - تطبيق التواصل الاجتماعي الشامل

## نظرة عامة
Elenty.app هو تطبيق تواصل اجتماعي شامل يجمع بين جميع وسائل التواصل في مكان واحد. يتيح للمستخدمين المشاركة، التواصل، والاتصال بطرق مبتكرة.

## المميزات الرئيسية

### 📱 التواصل الاجتماعي
- مشاركة الصور والفيديوهات
- القصص (Stories) التفاعلية
- التعليقات والإعجابات
- المتابعة والمتابعين

### 📞 المكالمات المتقدمة
- مكالمات صوتية عالية الجودة
- مكالمات فيديو HD
- مكالمات دولية
- سجل المكالمات

### 🎥 المحتوى المرئي
- فيديوهات قصيرة (Reels)
- البث المباشر
- التحديات والترندات
- المؤثرات البصرية

### 💬 المراسلة الفورية
- رسائل نصية
- مشاركة الوسائط
- الرسائل الصوتية
- المحادثات الجماعية

### 🎨 الإبداع والتخصيص
- مؤثرات الكاميرا
- فلاتر الجمال
- أدوات التحرير
- القوالب الجاهزة

## التقنيات المستخدمة

### Frontend
- **React Native** - إطار العمل الأساسي
- **Expo** - منصة التطوير
- **TypeScript** - لغة البرمجة
- **Zustand** - إدارة الحالة
- **React Query** - إدارة البيانات
- **Expo Router** - التنقل

### Backend (جاهز للتفعيل)
- **Node.js** - خادم التطبيق
- **PostgreSQL** - قاعدة البيانات
- **Redis** - التخزين المؤقت
- **Socket.io** - التواصل الفوري
- **WebRTC** - المكالمات المرئية

### الخدمات السحابية
- **AWS S3** - تخزين الملفات
- **CloudFront** - شبكة التوصيل
- **Firebase** - الإشعارات
- **Twilio** - المكالمات الدولية

## إعداد المشروع

### المتطلبات
- Node.js 18+
- Expo CLI
- React Native development environment

### التثبيت
```bash
# تثبيت المكتبات
npm install

# تشغيل التطبيق
npm start

# تشغيل على الويب
npm run start-web
```

### تفعيل Backend
1. انقر على "Backend" في القائمة العلوية
2. اختر نوع قاعدة البيانات (PostgreSQL موصى به)
3. قم بإعداد متغيرات البيئة
4. اتبع التعليمات لربط قاعدة البيانات

## هيكل قاعدة البيانات

### الجداول الرئيسية
- `users` - بيانات المستخدمين
- `posts` - المنشورات
- `stories` - القصص
- `messages` - الرسائل
- `chats` - المحادثات
- `calls` - المكالمات
- `notifications` - الإشعارات
- `challenges` - التحديات
- `effects` - المؤثرات
- `live_streams` - البث المباشر

### العلاقات
- علاقة واحد لمتعدد بين المستخدمين والمنشورات
- علاقة متعدد لمتعدد للمتابعة
- علاقة واحد لمتعدد للرسائل والمحادثات

## API Documentation

### Authentication
```
POST /api/v1/auth/login
POST /api/v1/auth/register
POST /api/v1/auth/logout
```

### Users
```
GET /api/v1/users/:id
PUT /api/v1/users/:id
GET /api/v1/users/search
```

### Posts
```
POST /api/v1/posts
GET /api/v1/posts/feed
POST /api/v1/posts/:id/like
POST /api/v1/posts/:id/comments
```

### Messages
```
POST /api/v1/chats/:id/messages
GET /api/v1/chats/:id/messages
GET /api/v1/users/:id/chats
```

### Calls
```
POST /api/v1/calls
GET /api/v1/users/:id/calls
PUT /api/v1/calls/:id/status
```
